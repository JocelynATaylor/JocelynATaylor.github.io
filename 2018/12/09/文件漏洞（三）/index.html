文件上传漏洞（三）

   补完剩下的…

Pass-13

查看代码，知道多了一个限制：
$bin = fread($file, 2); //只读2字节

即，上传的文件，程序将读取前两个字节，进行检查。
我们有两种方法：

1）文件头绕过检查

GIF89a:一个GIF89a图形文件就是一个根据图形交换格式（GIF）进行格式化之后的图形。

2）制作一句话木马图片：

   准备一张图片，一个包含一句话木马的php文件，和一个写了cmd的bat文件;

   点击bat文件，进入DOS，进行复制。




上传hc.jpg


Pass-14

关键语句：

   $info = getimagesize($filename);
   $ext = image_type_to_extension($info[2]);

getimagesize() 函数:
   用于获取图像大小及相关信息，成功返回一个数组，失败则返回 FALSE 并产生一条 E_WARNING 级的错误信息。

则第十三关的两种方法都可以采用。

Pass-15


注意，此题需要确保phpstudy里，使用的php版本配置中，开启了exif模块。否则，连正常的图片都无法上传。

开启php_exif模块的配置说明：

1.在php.ini文件中找到;extension=php_exif.dll，去掉前面的分号

2.在php.ini文件中找到;extension=php_mbstring.dll，去掉前面的分号，并将此行移动到extension=php_exif.dll之前。

3.找到[exif]段，把下面语句的分号去掉。

;exif.encode_unicode = ISO-8859-15

;exif.decode_unicode_motorola = UCS-2BE

;exif.decode_unicode_intel = UCS-2LE

;exif.encode_jis =

;exif.decode_jis_motorola = JIS

;exif.decode_jis_intel = JIS

则第十三关的两种方法都可以采用。
